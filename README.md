# 📚 Library App

A browser-based personal library tracker built with vanilla HTML, CSS, and JavaScript. Add books to your collection, track page counts, and mark them as read — no frameworks, no backend, no dependencies.

---

## ✨ Features

- **Add books** via a modal dialog form (title, author, number of pages)
- **Book cards** displayed in a grid, each showing title, author, page count, and read status
- **Remove books** from your library
- **Toggle read status** directly on each card
- No page refreshes — all interactions handled dynamically in the browser

---

## 🗂️ Project Structure

```
library_app/
├── index.html     # App markup, modal form, and card container
├── library.js     # Book object, library array, DOM rendering & event logic
└── style.css      # Card grid layout and styling
```

---

## 🛠️ Built With

- **HTML5** — semantic structure, native `<dialog>` element for the modal
- **CSS3** — card-based grid layout
- **Vanilla JavaScript** — object-oriented book model, dynamic DOM rendering

---

## 🚀 Getting Started

No installation needed. Just open the file in a browser:

```bash
# Clone the repository
git clone https://github.com/ehab-walid/library_app.git
cd library_app

# Open in your browser
open index.html
```

Or download the ZIP from GitHub and open `index.html` directly.
